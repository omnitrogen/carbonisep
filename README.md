# carbonisep

Triangle des Bermudes

## Installation

Installer node et npm.

Sur un terminal:

```
cd server
npm i
npm start
```

Sur <http://localhost:8000/hello> le message de test devrait apparaitre.

Dans un autre terminal, faire:

```
cd client
npm i
npm start
```

L'application devrait s'ouvrir sur <http://localhost:3000>.
